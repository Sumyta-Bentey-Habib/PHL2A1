# Assignment 01 - TypeScript Learning

A comprehensive TypeScript assignment covering fundamental concepts and practical problem-solving.

## 📁 Project Structure

```
assignment-01/
├── blog-1.md          # TypeScript Type Safety Guide
├── blog-2.md          # OOP Pillars in TypeScript
├── solutions.ts       # Problem Solutions
└── README.md          # This file
```

## 📚 File Descriptions

### `blog-1.md` - TypeScript Type Safety Explanation
A guide explaining type safety in TypeScript with practical examples:
- **`any` vs `unknown`**: Understanding the differences between these types
- **Type Narrowing**: Techniques for determining and narrowing variable types
- Includes real code examples demonstrating type safety best practices

### `blog-2.md` - OOP Pillars in TypeScript
An educational document covering the four fundamental pillars of Object-Oriented Programming:
1. **Encapsulation** - Data hiding and access control
2. **Inheritance** - Code reuse through class extension
3. **Polymorphism** - Method overriding for different behaviors
4. **Abstraction** - Interface-based programming

Each pillar includes code templates and practical examples.

### `solutions.ts` - Problem Solutions
TypeScript solutions to 7 programming challenges:

1. **filterEvenNumbers** - Filters an array to return only even numbers
2. **reverseString** - Reverses a string character by character
3. **checkType** - Determines if a value is a string or number
4. **getProperty** - Generic function to safely access object properties
5. **toggleReadStatus** - Adds a property to an object using type intersection
6. **Student Class** - Inheritance example with Person parent class
7. **getIntersection** - Finds common elements between two arrays
